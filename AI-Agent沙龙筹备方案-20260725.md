# 「AI Agent 落地实战」线下沙龙筹备方案

- **时间**：2026-07-25（周六）14:00–17:30（13:30 开始签到）
- **地点**：上海（场地待最终选定，见场地模块）
- **规模**：30 人（观众约 24 + 嘉宾 3–4 + 工作人员 2–3）
- **总预算**：8000 元
- **形式**：3 场主题分享（40 分钟/场）+ 圆桌交流 + 茶歇 networking

---

## 一、分模块 Checklist

### ① 场地（负责人：____）
- [ ] 确定场地：联合办公空间 / 咖啡馆包场 / 企业会议室赞助，30–35 人剧院式布局，周六下午半天
- [ ] 签协议并付定金，留场地联系人电话
- [ ] 确认硬件：投影或 LED 屏、音响 + 无线麦 ×2、翻页笔、Wi-Fi（30 人同时在线）、空调
- [ ] 确认能否外带茶歇、是否收服务费/清洁费
- [ ] 确认物业要求：外来人员登记、电梯刷卡、指示牌可否张贴
- [ ] D-1 踩点：实测投影、话筒、翻页笔、插座位置，拍照记录签到处位置
- [ ] 交通信息整理（地铁/停车），写进参会通知

### ② 餐饮（负责人：____）
- [ ] 预定茶歇 30 人份：咖啡/茶/软饮 + 点心 + 水果（约 40 元/人），约定 15:00 前送达
- [ ] 瓶装水 35 瓶，签到时放置
- [ ] 素食/清真备选各 2 份（报名时问卷收集饮食禁忌）
- [ ] 工作人员简餐 5 份
- [ ] 确认垃圾清运方式，备垃圾袋
- [ ] D-1 电话二次确认送达时间和数量

### ③ 宣传（负责人：____）
- [ ] 定稿主视觉海报：主题、时间地点、议程、嘉宾、报名二维码
- [ ] 活动行/互动吧发布报名页：免费票 + 审核制，**放票 38 张**（到场率按 75%–80% 估，保 30 人到场）
- [ ] 报名表字段：姓名、公司、职位、微信号、对 AI Agent 落地最关心的问题（供嘉宾备讲）
- [ ] 首轮投放（D-5）：朋友圈、10+ 技术社群、公众号/即刻/小红书
- [ ] 中期盘点（D-3）：报名 < 25 → 二次推送 + 定向邀请补位
- [ ] D-2：截止报名，发参会通知（时间/地点/交通/群二维码），建活动微信群
- [ ] 群内预热：发议程 + 嘉宾介绍

### ④ 物料（负责人：____）
- [ ] 易拉宝 ×2（门口指引 + 场内背景）
- [ ] 签到表按报名名单打印 ×2 份 + 签字笔
- [ ] 胸牌/名牌贴纸 38 份（观众/嘉宾/工作人员三色区分）
- [ ] 反馈问卷：在线版生成二维码（主用），打印 5 份备用
- [ ] 设备类：翻页笔、HDMI/Type-C 转接头、插线板 ×2、A4 纸、马克笔、胶带剪刀
- [ ] 嘉宾伴手礼/交通答谢（现金红包或等价礼品，含在嘉宾预算）
- [ ] D-2 前全部到货清点，D-1 打包成「现场物料箱」

### ⑤ 现场流程（负责人：____）
**当天时间表**
| 时间 | 内容 |
|---|---|
| 12:30–13:30 | 工作人员到场：布置签到台、贴易拉宝、调试设备、摆水 |
| 13:30–14:00 | 签到、发名牌、自由交流 |
| 14:00–14:10 | 主持人开场（介绍议程、合影预告、问卷二维码露出） |
| 14:10–14:50 | 分享一（30 分钟讲 + 10 分钟 Q&A） |
| 14:50–15:30 | 分享二 |
| 15:30–15:50 | 茶歇 + 集体合影 |
| 15:50–16:30 | 分享三 |
| 16:30–17:10 | 圆桌：嘉宾 + 观众提问（主持人备 3 个引导问题） |
| 17:10–17:20 | 抽奖/问卷填写/收尾致谢 |
| 17:20–17:40 | 自由 networking |
| 17:40–18:00 | 撤场：物料回收、垃圾清理、场地检查 |

**分工（2–3 人可兼任）**
- 签到接待 ×1、设备/PPT 切换 ×1、主持 ×1、摄影（可志愿者）、机动 ×1

**会后动作**
- [ ] D+1：群内发感谢公告 + 精选照片
- [ ] D+3：recap 推文/笔记发布，问卷数据统计，费用结算归档

### ⑥ 应急（负责人：____）
- [ ] PPT 三保险：嘉宾提前 D-2 交稿 → 备用电脑 + U 盘 + 网盘各存一份
- [ ] 网络故障：手机热点备用；PPT 全部本地化，不依赖在线演示
- [ ] 话筒/投影故障：场地联系人电话打印贴在控制台；备小蜜蜂扩音器 1 个
- [ ] 嘉宾缺席：备 1 个自己人可顶上的备胎议题，或当场延长圆桌
- [ ] 天气：7 月上海高温 + 台风季，D-2 查天气预报，异常则群内发交通/防暑提醒
- [ ] 医疗：创可贴、藿香正气水，查好最近药店/医院路线
- [ ] 人数超预期：预留加座 5 个的空间；人数不足：D-3 启动定向邀请
- [ ] 突发开支：从备用金支出，单笔 > 300 先电话确认
